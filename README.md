# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** game built using **HTML5**, **CSS3**, and **JavaScript**. This project allows two players to play the classic game in a clean and responsive user interface.

---

## 📖 Overview

This project is a browser-based implementation of the classic Tic Tac Toe game. It was developed to practice DOM manipulation, JavaScript logic, event handling, and responsive web design.

---

## ✨ Features

- 🎯 Two-player gameplay (Player X vs Player O)
- 🏆 Automatic winner detection
- 🤝 Detects draw/tie situations
- 🔄 Restart/New Game functionality
- 📱 Responsive design for desktop and mobile devices
- 🎨 Clean and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)


---

## 🎮 How to Play

1. The game starts with **Player X**.
2. Players take turns clicking on an empty cell.
3. The first player to align **three symbols** horizontally, vertically, or diagonally wins.
4. If all cells are filled without a winner, the game ends in a draw.
5. Click **Restart** or **New Game** to play again.
